# 🛒 CoopShop Sales Management System Analysis

Business Analysis and System Analysis project for an online sales management system. This project focuses on requirement analysis, business process modeling, system design, and UI prototyping.

---

## 📌 Project Overview

CoopShop is an online retail management system that supports customers in searching products, placing orders, making payments, and tracking deliveries while enabling administrators to manage products, orders, and customer information.

---

## 🎯 My Responsibilities

- Gathered and analyzed business requirements.
- Identified functional and non-functional requirements.
- Developed Context Diagram, DFD Level 0, and DFD Level 1.
- Created Use Case Diagrams and Use Case Specifications.
- Designed ERD and Data Dictionary.
- Designed UI wireframes.
- Collaborated with team members during system analysis and design.
